<objective>
Fix the runtime panic that prevents bench_online_p1 and bench_online_p2 from producing
any measurement.

**Root cause** (from 10-VERIFICATION.md and 10-REVIEW.md CR-01): the online benches build
their generator and evaluator with two independent helpers — setup_auth_gen (line 52) and
setup_auth_eval (line 59) — each of which calls TensorFpre::into_gen_eval(). That function
explicitly leaves gamma_d_ev_shares as vec![] (documented at src/auth_tensor_fpre.rs:180-183
and 194-197). Both assemble_c_gamma_shares (src/lib.rs:109) and assemble_c_gamma_shares_p2
(src/lib.rs:218-219) assert gamma_d_ev_shares.len() == n * m. These asserts panic on the
first benchmark iteration of every (n, m) pair. cargo bench --no-run exits 0 (compilation
is clean), but no measurement can run.

**Fix:** add a single new helper setup_auth_pair(n, m, chunking_factor) -> (AuthTensorGen,
AuthTensorEval) that uses IdealPreprocessingBackend.run(n, m, 1, chunking_factor) — which
already populates gamma_d_ev_shares (and the other D_ev fields) with n*m / n / m correlated
authenticated bits (verified by tests test_ideal_backend_gamma_d_ev_shares_length and
test_ideal_backend_d_ev_shares_lengths in src/preprocessing.rs:361-405). Replace the two
separate setup calls at benches/benchmarks.rs:176-177 (P1) and 287-288 (P2) with a single
destructured call to setup_auth_pair.

**Do NOT** remove setup_auth_gen or setup_auth_eval. They remain correct for
bench_online_with_networking_for_size (lines 363-418), which never calls
assemble_c_gamma_shares / assemble_c_gamma_shares_p2 and therefore does not need a
correlated pair. Removing them would break the network benchmark group.

Purpose: closes the BENCH-02 / BENCH-04 / BENCH-06 partial-satisfaction gap from
10-VERIFICATION.md, allowing the online benchmarks to actually execute and produce wall-clock
measurements. No production code (src/) is touched.

Output: a single edit to benches/benchmarks.rs that adds the import line, the helper
function, and replaces two pairs of setup calls.
</objective>
